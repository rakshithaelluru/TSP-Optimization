# TSP Optimization using GA, PSO, ACO

## 📌 Description
This project solves the Travelling Salesman Problem (TSP) using:
- Genetic Algorithm (GA)
- Particle Swarm Optimization (PSO)
- Ant Colony Optimization (ACO)

## 📂 Project Structure
- code/ → Python implementation
- report/ → Project report

## ▶️ How to Run

pip install numpy matplotlib  
python code/ot_project.py

## 📊 Output
- Displays best distance for GA, PSO, ACO
- Shows graph of optimal path
